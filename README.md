## 🔥 About "Trends"

**Trends** is an open-source Android client for **Google Trends** website. It is a multi-module project based on Kotlin MVI architecture and uses Jetpack compose to implement its UI.

## 🏗 Tech Stack
- 100% Jetpack Compose UI
- Kotlin Coroutines
- Flow
- DataStore
- Hilt
- Coil
- Retrofit

## 🔬 Automated test
- Junit
- Mockk
- Espresso
